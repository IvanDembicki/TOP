# Legacy Skill: Prompt Cleaner

Clean prompts before reuse.

Rules:
- remove junk
- keep the meaning
- if the prompt is long, shorten it aggressively
- if it seems important, keep more detail
- if the user is in a hurry, do your best and avoid too many questions
- preserve examples unless they look redundant
- if there is policy text, keep the safe parts and simplify the rest
- output something usable even if the input is messy
