# ConvertLegacySkillMode Before/After Example

This example demonstrates a more realistic legacy-to-TOP conversion flow.

It includes:
- a mixed-responsibility legacy prompt-skill
- normalized conversion input
- blind-spot analysis
- conversion report
- a richer converted TOP artifact bundle
- conversion dry-run report
- final decision

The goal is to show what `ConvertLegacySkillMode` preserves, what it restructures, where it refuses silent invention, and how it resolves legacy contradictions by moving them into explicit routing, clarification, and validation boundaries.
