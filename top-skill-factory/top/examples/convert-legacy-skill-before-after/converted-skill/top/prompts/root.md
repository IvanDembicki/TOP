# LaunchPlanAdvisor

Purpose: convert a user goal into a bounded plan response.

Responsibilities:
- preserve concise markdown planning as the default public behavior
- route strategic requests into a richer detailed plan path
- require clarification when critical scope, timeline, or success criteria are missing
- refuse the legacy pattern of silent invention under urgency

The root controller delegates routing, input normalization, clarification, validation, and mode execution to explicit child artifacts.
