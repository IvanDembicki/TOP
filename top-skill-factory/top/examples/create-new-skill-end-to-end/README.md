# CreateNewSkillMode End-to-End Example

This example demonstrates one bounded end-to-end path through `CreateNewSkillMode`.

It includes:

- the original user request
- a normalized input artifact
- a compact generated skill bundle
- a dry-run report
- a final decision artifact

Purpose of this example:

- show what a generated skill bundle can look like
- show how validation and dry run affect readiness
- show that the output is an artifact set, not just prose

This is a small demonstration bundle, not a claim that every generated skill must have exactly this size or file layout.
This example uses the minimal_demo_contract declared in generated-skill/top/artifact-manifest.json.
