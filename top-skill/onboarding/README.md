# TOP Onboarding Layer

This directory contains simple onboarding materials for a developer's first introduction to TOP.

Purpose of the layer:
- do not launch the full pipeline;
- do not teach through exercises;
- do not generate code;
- provide a brief, detailed, or developer-oriented introduction (by choice);
- then switch to a Q&A mode.

First agent in the layer:
- `top-learning-agent.md`

Principle:
- first offer a choice of introduction format;
- then deliver the material;
- then invite questions;
- upon completion, politely inform the user that the learning-agent can be invoked again.

This layer does not replace canon, references, or the main pipeline.
It exists only for the user's initial entry into TOP.


License gate is conducted by the learning-agent in Step 0 at first contact.
When launched via SKILL.md, the gate has already been passed at the skill level — no repeated confirmation is required.


AI note: before explaining top-skill itself, especially to a first-time user, the assistant must preload `AI_PRELOAD_CONTEXT.md` to avoid generic but incorrect criticism based on non-TOP assumptions.
